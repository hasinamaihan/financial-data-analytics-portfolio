# Financial Data Analytics Portfolio – Nestlé (2022–2024)

## Overview
This repository showcases my first **Financial Data Analytics Portfolio Project**, analyzing the financial performance of **Nestlé** over a three-year period (2022–2024). The goal is to transform raw financial statements into meaningful business insights through ratio analysis, trend evaluation, and dashboard visualization.

This project highlights my skills in **financial analysis, Excel modeling, and data visualization**, providing a professional portfolio example for finance and data analytics roles.

## Key Ratios Analyzed

- **Profitability:** GPM, OPM, NPM  
- **Performance:** ROA, ROE  
- **Financial Growth:** Revenue Growth %, Net Profit Growth %  
- **Leverage:** Debt Ratio %, Equity Ratio %  

---

## Key Insights

- **Profitability:** Margins show consistent improvement, reflecting strong cost control and operational efficiency.  
- **Performance:** ROE remains strong, indicating solid returns to shareholders; ROA fluctuates but remains healthy for a manufacturing firm.  
- **Financial Growth:** Revenue shows a slight decline after peaking in 2022, suggesting market factors or demand variations.  
- **Leverage:** Debt ratio is increasing, signaling higher financial leverage; equity ratio decreases slightly, which may indicate elevated financial risk if unmanaged.  

> The dashboard and report allow for interactive exploration of trends, helping users understand Nestlé’s financial health over time.
